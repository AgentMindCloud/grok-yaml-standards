# CONTRIBUTING.md

Thank you for helping make Grok YAML standards the official community reference!

### How to Contribute
1. Fork the repo
2. Create a branch: `feat/grok-agent-improvement`
3. Update the relevant `example.yaml`, `schema.md`, or add new use cases
4. Ensure every change is forward-compatible with `grok-install.yaml`
5. Open a PR with clear description

**We welcome**:
- New real-world use cases
- Security hardening suggestions
- Schema improvements
- Translations

All contributors will be credited in the next release and on X.
