## Want to be listed?

Open any PR against [`AgentMindCloud/grok-yaml-standards`](https://github.com/AgentMindCloud/grok-yaml-standards) — every merged contribution gets a row in this table and a mention in the release notes on the next tag.

Good first issues:

- Add a real-world use case to any `grok-*/usecases.md`.
- Propose a new trigger example in `grok-*/x-trigger-examples.md`.
- Improve a JSON Schema (tighter enums, better descriptions, more examples).
- Fix a typo. No PR is too small.

See [`CONTRIBUTING.md`](CONTRIBUTING.md) for the branching convention and review flow.
