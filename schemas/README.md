# schemas/ Folder – JSON Schema Validation

This folder contains official JSON Schema files for **all 8 magic YAML files**.

**How to use:**
- GitHub automatically validates any `.grok/*.yaml` file against these schemas
- VS Code / Cursor / any IDE with YAML support will give you auto-complete and error checking
- Perfect for contributors and xAI adoption

**Files included:**
- grok-config.json
- grok-prompts.json
- grok-agent.json
- grok-workflow.json
- grok-update.json
- grok-test.json
- grok-docs.json
- grok-security.json

**Version:** 1.2.0  
**Forward-compatible** with `grok-install.yaml`
